## Sam Altman：走向智能时代——2026年度展望

> [原文连接](https://blog.samaltman.com/the-intelligence-age) 2026年1月

------

> OpenAI CEO Sam Altman 在 2026 年初发表了年度展望长文，正式宣告人类进入"智能时代"。在 GPT-5 和 o3 系列模型相继落地、AI Agent 开始大规模替代知识工作流程之后，他以更坚定的语气描述了他对接下来五年的判断。

------

我想谈谈一件已经开始发生、但我们还没有完全消化的事：**人类正在进入一个全新的时代。**

不是"AI 辅助时代"，不是"AI 增强时代"，而是真正的**智能时代**——AI 系统开始在越来越多的领域表现出超越专家级别的能力，并且开始能够自主完成需要多步推理、长时间规划的复杂任务。

## 从工具到 Agent

过去几年，AI 是一个工具。你给它一个问题，它给你一个答案。这仍然很有价值，但本质上和计算器、搜索引擎没有区别。

2026 年正在发生的变化是：AI 开始成为 Agent——一个可以自主执行任务、调用工具、迭代改进、与其他 Agent 协作的系统。

区别是巨大的：
- **工具**：帮你写一封邮件；
- **Agent**：帮你管理收件箱、起草回复、跟进事项、在日历上安排会议；
- **更强的 Agent**：帮你完成一个季度的市场调研，自主搜集数据、撰写报告、验证结论。

这不是科幻。这正在发生。

## 增长的速度超出了所有人的预期

我在 2023 年写过，我们对 AI 能力增长速度的预测往往偏低。三年过去了，这个规律仍然成立。

最让我感到惊讶的，不是 AI 在代码、数学、写作上的能力——那些我们大致预期到了。让我惊讶的是 AI 的**泛化能力**：一个没有被专门训练过医学的模型，在某些诊断任务上开始超越专科医生；一个没有被专门训练过法律的模型，在某些案例分析上开始超越律师。

这说明，我们训练出来的不只是一个更好的语言模型。我们可能正在接触某种更本质的东西。

## 最重要的问题：收益如何分配

如果 AI 能让每个工人的生产力提升十倍，会发生什么？

乐观的情景：财富大幅增加，工作性质改变，人们有更多时间做创造性的事情。
悲观的情景：财富集中在少数人手中，大量工作岗位消失，社会不平等加剧。

我认为这是 2026 年最重要的政策问题，比 AI 安全本身更紧迫。AI 安全是一个长期问题；AI 带来的经济冲击是一个正在发生的问题。

OpenAI 的使命是让 AGI 造福全人类。这不只是一句话，而是意味着我们必须在能力研发之外，认真思考分配问题。

## 给创业者的判断

**现在是在 AI 之上创业的最好时机，没有之一。**

原因：
1. 底层能力（推理、代码、多模态）已经足够强，可以支撑真正有价值的应用；
2. 大多数行业还没有被重新设计，先入者优势明显；
3. AI Agent 框架逐渐成熟，可以用更小的团队完成更复杂的产品。

但有一点需要警惕：**不要把"用了 AI"当作护城河**。调用同一个 API 的公司有数万家。真正的护城河来自：对特定场景的深度理解、独特的数据资产、和用户建立的信任关系。

**_"未来五年，会有人用 AI 重新发明每一个行业。这些人可能已经在路上了。问题是：你是他们中的一员吗？"_**
